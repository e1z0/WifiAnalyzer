# Wifi Analyzer

It's based on https://github.com/moononournation/ESP8266WiFiAnalyzer with modifications to support TFT 480x320 RaspberryPI LCD Screen

# Tested with these esp8266 boards

* LOLIN(WEMOS) D1
* Amica NodeMCU
